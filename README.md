## 项目的运行逻辑  
  
1. **初始屏幕检测**：检测`start_image.png`。  
2. **加载AI模型**：尝试读取`game_ai_model.pth`，如不存在则创建新模型。  
3. **启动线程**：处理游戏逻辑、图像识别、AI决策。  
4. **游戏控制**：  死亡后`j`键重启游戏。  
5. **停止训练**：检测`end_game_image.png`停止训练。（仅限游戏中，死亡时无法检测）
